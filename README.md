# 🧠 Quiz App (UI)

This is a simple Quiz Application user interface created using HTML and CSS.

 📌 Features
- Clean and structured quiz layout
- Multiple choice question design
- Styled buttons and options
- Responsive UI for different screen sizes

 🛠️ Tech Stack
- HTML
- CSS

🚀 Purpose
This project was built to practice front-end development skills like layout design, styling, and structuring a quiz interface.
